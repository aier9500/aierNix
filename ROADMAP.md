## Mission

Transfer the `tuxies-wiki` **aiers-fedora-checklist.md** setup into declarative Nix configs.
Repo is **device-portable**: `default/hardware-configuration.nix` is gitignored and generated
per-machine. Launch goal: config builds + boots in a **VM** now, with the portable mindset kept
so the same flake later runs on baremetal.
